# iterblock2
A wordlist file generator for bruteforcing and dicationary attacks.

Example usage:

```
python3 iterblock.py '1!2@3#4$5%6^7&8*9(0)-_=+qQwWeErRtTyYuUiIoOpP[{]}\|aAsSdDffgGhHjJkKlL;:'""'zZxXcCvVbBnNmM,<.>/?' 6
> /mnt/wordlists/6_large.out &
```
